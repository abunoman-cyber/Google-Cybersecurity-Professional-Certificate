# Cloud Computing and SDNs in Modern Networking

Traditional on-premise networks are changing. Most organizations now move to the cloud to save costs and improve security. 

### Cloud Service Models:
* **SaaS:** Using ready-made software over the internet.
* **IaaS:** Renting virtual servers and storage (the digital backbone).
* **PaaS:** Tools for developers to build custom apps without managing hardware.

### Hybrid and Multi-Cloud:
Most companies don't just use one system. They use a **Hybrid** model (On-premise + Cloud) or **Multi-cloud** (using multiple providers like Google and AWS) to keep their data safe and redundant.

### Software-Defined Networking (SDN):
In the cloud, we don't touch physical wires. SDNs allow us to use virtual switches, routers, and firewalls. 
* **Key Benefits:**
  1. **Scalability:** We only pay for what we use (Elastic model).
  2. **Reliability:** Services are almost always up and running.
  3. **Security:** I can deploy a Web Application Firewall (WAF) or IDS/IPS instantly through a web console to stop threats.

My takeaway: Cloud computing isn't just about storage; it's about being able to scale security and infrastructure at the speed of software.
