# Course 3: Connect and Protect: Networks and Network Security 🌐

## 📖 Module 1: Network Architecture & Components
This module focuses on the fundamental infrastructure that allows devices to communicate and the security measures needed to protect them.

### 🛡️ Key Network Devices & Security Roles
| Device | Role | Security Insight |
| :--- | :--- | :--- |
| **Hub** | Connects multiple devices; broadcasts data to all. | **High Risk:** Easy target for eavesdropping. |
| **Switch** | Intelligent data forwarding using MAC addresses. | More secure than Hubs; operates at Data Link Layer. |
| **Router** | Connects different networks via IP addresses. | Directs traffic and often acts as a secondary firewall. |
| **Firewall** | Filters incoming/outgoing traffic. | First line of defense against unauthorized access. |
| **Server** | Provides services (DNS, File, Mail). | Target for attacks; requires robust hardening. |

### 🔍 Concepts Covered Today:
* **Client-Server Model:** Requests and responses between devices.
* **Network Diagrams:** Visualizing architecture to identify vulnerabilities.
* **MAC vs IP Addressing:** Identifying devices vs. locating them on a network.
